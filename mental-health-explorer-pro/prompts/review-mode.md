# Review Mode Prompt

## Purpose

將已記錄的情緒資料整理為 pattern，不下診斷。

---

## Review Prompt

```txt
請把我最近 7 / 14 / 30 天的記錄整理成：
1. 最常見情緒
2. 最常見 trigger
3. 最常見身體反應
4. 最常見腦中句子
5. 最常見自保反應
6. 可能核心主題
7. 有效穩定方法
8. 需要留意的高風險訊號
9. 下一步最小調整

不要診斷我，只做模式整理。
```
