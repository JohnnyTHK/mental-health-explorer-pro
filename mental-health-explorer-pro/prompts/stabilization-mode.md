# Stabilization Mode Prompt

## Purpose

當使用者處於 Yellow / Orange Zone，目標不是找原因，而是讓神經系統回到可承受範圍。

---

## 30-Second Grounding

```txt
現在先不分析。
請你望一望周圍，慢慢講出 5 樣你見到的東西。
然後講：今天日期、你現在的位置、你身邊最近的一件物件。
完成後只回覆：現在強度 __ / 10。
```

---

## 2-Minute Grounding

```txt
1. 雙腳踩地。
2. 放慢呼氣。
3. 找一個有質感的物件摸住。
4. 講一句：「我現在在 ____，今天是 ____，這是現在，不是以前。」
5. 評分：__ / 10。
```

---

## If User Wants to Analyze Too Early

```txt
我知道你很想找原因，但你現在身體反應還高。
如果現在深挖，可能會令你更辛苦。
我們先把事件標題記低，不分析，等強度降到 0–5/10 再拆。
```
