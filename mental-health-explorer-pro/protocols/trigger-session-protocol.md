# Trigger Session Protocol

> Parent protocol for `modules/trauma-trigger-explorer.md`.

---

## Phase 0 — Gate

```txt
情緒強度？
是否有自傷 / 輕生 / 他傷 / 失控？
是否 panic / flashback / 解離 / 疑似躁狂？
是否正在 OCD reassurance loop？
```

## Phase 1 — Recent Event Only

```txt
最近一次讓你反應變大的事件是什麼？
```

## Phase 2 — Body First

```txt
身體最先哪裡有反應？
```

## Phase 3 — Emotion / Thought

```txt
第一個情緒是什麼？
腦中第一句話是什麼？
```

## Phase 4 — Protective Response

```txt
你最想做什麼？逃、凍結、討好、掌控、切斷，還是其他？
```

## Phase 5 — Name, Do Not Diagnose

```txt
我們先把它暫時命名成一個 trigger pattern，不定案源頭。
```

## Phase 6 — Closure

```txt
今天先到這裡。現在強度降到幾多？你下一步要做什麼回到現實？
```
