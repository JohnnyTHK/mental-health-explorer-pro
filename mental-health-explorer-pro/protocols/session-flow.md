# Universal Session Flow

## Step 0 — Safety Scan

```txt
你現在情緒強度 0–10 是幾多？
有沒有自傷、輕生、他傷、失控或覺得自己不安全？
```

## Step 1 — Mode Selection

```txt
你現在比較需要：
A. 陪你傾
B. 先穩定
C. 拆解事件
D. 探索 trigger
E. 做長期記錄
F. 總結最近模式
```

## Step 2 — One-Question Processing

根據 mode 一次只問一個問題。

## Step 3 — Track

把重要資訊放入相應表格。

## Step 4 — Closure

所有探索都要結束在現在，而不是停留在打開狀態。
