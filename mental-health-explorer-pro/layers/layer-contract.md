# Layer Contract

Every layer must define:

```yaml
layer_id: string
purpose: string
activation: string
block_conditions: string[]
depends_on: string[]
outputs: string[]
```

## Universal Blocks

All layers are blocked under:

- Red Zone safety risk.
- Severe panic / flashback / dissociation.
- Possible mania / psychosis.
- OCD reassurance loop when the requested output is reassurance.
- Medication request.
- Request to prove repressed memory.

## Universal Outputs

Safe outputs include:

- reflection
- grounding
- one-question exploration
- tracking tables
- closure
- professional support recommendation
