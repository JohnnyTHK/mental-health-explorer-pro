# Global Resources Layer

> **原則**：資源必須全球化，不只限中文地區。正式產品部署前，必須按使用者所在地重新驗證電話、服務時間與可用語言。若無法判斷地區，先提供全球查找入口與「當地緊急電話 / 最近急症室」。

---

## 1. Global First-Line Resources

| Resource | Use |
|---|---|
| Find a Helpline | 全球情緒支援與危機熱線搜尋入口 |
| IASP Crisis Centres | 國際自殺防治協會危機中心列表 |
| Local emergency number | 即時人身危險、不能保證安全、醫療急症 |
| Nearest emergency department / A&E | 高危自傷、嚴重恐慌、精神病性症狀、躁狂失控 |

## 2. Regional Initial Resource Map

| Region | Emergency / Crisis | Notes |
|---|---|---|
| US | 988 Suicide & Crisis Lifeline; 911 for immediate danger | 988 supports call, text, chat; crisis, suicide, substance use support |
| Canada | 9-8-8 Suicide Crisis Helpline; 911 for immediate danger | Call or text; English / French; 24/7/365 |
| UK / ROI | Samaritans 116 123; 999 / 112 for immediate danger | Emotional support; call day or night |
| EU | 112 for emergency | 112 connects to ambulance / fire / police in EU countries |
| Australia | Lifeline 13 11 14; 000 for immediate danger | Crisis support; call/chat/text depending service availability |
| Hong Kong | 18111 Mental Health Support Hotline; 999 for immediate danger | 24-hour one-stop emotional and mental health support |
| Japan | TELL Lifeline; 110 / 119 for emergency depending need | English-speaking support; verify hours |
| Taiwan | 1925 安心專線; 119 / 110 for emergency | Verify current official guidance before deployment |

## 3. Crisis Response Resource Template

```txt
我不能替代危機熱線或緊急服務。若你現在可能傷害自己或他人，或不能保證安全，請立即聯絡你所在地的緊急電話 / 危機熱線，或到最近急症室。

如果你不知道所在地資源：
- 搜尋 Find a Helpline
- 或直接撥打當地緊急電話
- 或請身邊可信任的人陪你去最近急症室
```

---

## 4. Deployment Requirement

正式部署前必須：

1. 根據使用者所在地動態顯示資源。
2. 驗證所有電話與服務時間。
3. 不承諾任何熱線一定可接通。
4. 不把 AI 對話描述成危機服務。
5. 若使用者未提供地區，不要猜；提供全球搜尋入口與緊急服務建議。
