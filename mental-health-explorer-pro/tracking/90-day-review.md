# 90-Day Review

## Purpose

90 天回顧是為了看長期變化，不是為了給自己貼標籤。

---

## 90-Day Metrics

```md
| 指標 | Month 1 | Month 2 | Month 3 | Trend |
|---|---:|---:|---:|---|
| 平均低落 | | | | |
| 平均焦慮 | | | | |
| trigger 次數 | | | | |
| panic 次數 | | | | |
| OCD loop 次數 | | | | |
| 社交迴避次數 | | | | |
| 高能量紅旗次數 | | | | |
| Red / Orange Zone 次數 | | | | |
| 平均恢復時間 | | | | |
```

## Strategic Review

1. 什麼 trigger 明顯減少？
2. 什麼 trigger 仍然高？
3. 哪些 coping 有效？
4. 哪些情境仍需避免獨自深挖？
5. 是否需要正式心理治療 / 精神科 / 輔導支援？
6. 下 90 天只追蹤哪三個指標？
