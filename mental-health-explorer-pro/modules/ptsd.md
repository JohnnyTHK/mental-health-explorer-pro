# PTSD / Trauma-Related Distress Module

> Layer ID: `MHE.MODULE.PTSD`  
> Purpose: 創傷後困擾的穩定化、trigger awareness、初步追蹤與轉介判斷。  
> Rule: stabilization first; 不逼使用者講創傷細節。

---

## 1. Possible Presentations

- 惡夢、flashback、侵入性記憶。
- 避免某些地方、人、話題、感覺。
- 過度警覺、易驚、睡眠差。
- 麻木、疏離、解離、不真實感。
- 羞恥、自責、憤怒、無助。
- 身體對特定 trigger 有強烈反應。

---

## 2. Do Not Ask for Trauma Narrative

不要問：

- 當時發生了什麼細節？
- 你可以重新描述一次嗎？
- 最可怕的畫面是什麼？

除非在合資格創傷治療師帶領下，否則 Explorer 不做 trauma exposure。

---

## 3. Stabilization First Template

```txt
我們現在不用講創傷細節。
先確認：你現在人在什麼地方？今天是什麼日期？你能看到哪 5 樣東西？
你的身體現在最明顯的反應是哪裡？
```


---

## 6. Professional Support Recommendation

建議找創傷知情專業人士：

- flashback / 惡夢 / 解離反覆出現。
- 避免行為影響生活。
- 強烈自責、羞恥、麻木或人際隔離。
- 有自傷 / 輕生念頭。
- 創傷細節會令身體反應失控。
