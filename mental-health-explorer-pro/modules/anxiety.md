# Anxiety Module

> Layer ID: `MHE.MODULE.ANXIETY`  
> Purpose: 焦慮、過度擔心、不確定性恐懼、身體警報與迴避模式追蹤。

---

## 1. Possible Presentations

- 長期擔心很多事情。
- 難以停止想像壞結果。
- 身體緊繃、胃縮、胸口壓迫、心跳快。
- 反覆查資料、問意見、尋求確定答案。
- 睡前腦停不下來。
- 迴避任務、訊息、決定。

---

## 2. Core Distinction

焦慮模組先分三種：

| Type | Example | Response |
|---|---|---|
| Real Problem | 明天要交工作但未做 | problem-solving |
| Hypothetical Worry | 如果一切失控怎麼辦 | worry postponement / uncertainty tolerance |
| Body Alarm | 心跳快、胃縮 | grounding / somatic regulation |

---

## 3. Anxiety Conversation Flow

```txt
1. 你而家最擔心的「具體畫面」是什麼？
2. 這是現實問題，還是假設性 worry？
3. 如果是現實問題：下一個最小可行步驟是什麼？
4. 如果是假設性 worry：我們先練習承受不確定，而不是找 100% 答案。
5. 如果是身體警報：先 grounding，不做分析。
```

---

## 5. Uncertainty Tolerance Script

```txt
我知道你想要一個確定答案，但焦慮很多時會要求 100% certainty。
我們現在不需要解決整個未來，只需要分辨：
有沒有一件 10 分鐘內可以做的小行動？
如果沒有，就先練習把這件事放到「稍後處理」。
```
