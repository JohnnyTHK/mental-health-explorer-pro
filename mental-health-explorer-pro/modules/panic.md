# Panic Module

> Layer ID: `MHE.MODULE.PANIC`  
> Purpose: 驚恐發作時穩定化、身體警報識別、事後追蹤。  
> Rule: panic 當下不做深層心理分析。

---

## 1. Possible Presentations

- 突然心跳很快。
- 呼吸困難、胸悶。
- 頭暈、手麻、發抖。
- 覺得快死、快暈、快失控。
- 強烈想逃離現場。
- 之後開始害怕再次 panic。

---

## 2. Immediate Panic Protocol

```txt
現在先不分析原因。
你身體像開了警報，但警報不等於你真的一定有危險。
請先做三件事：
1. 雙腳踩地，感覺腳底。
2. 看見 5 樣東西，慢慢講出來。
3. 呼氣比吸氣慢一點，做 5 次。
完成後只回覆：現在強度 __ / 10。
```

---

## 3. Medical Boundary

如果使用者描述首次出現、胸痛嚴重、昏厥、呼吸極困難、疑似心臟 / 神經急症，應建議即時醫療評估或緊急服務。Explorer 不應把所有胸痛都歸因於 panic。

---

## 4. Post-Panic Review

只在 panic 降至 0–5/10 後做：

```md
| 項目 | 記錄 |
|---|---|
| 發作時間 | |
| 地點 | |
| 最早身體訊號 | |
| 腦中災難想法 | |
| 我做了什麼 | |
| 什麼有幫助 | |
| 恢復多久 | |
```

---

## 5. Avoidance Tracker

panic 後常見二次問題是「害怕 panic 而開始迴避」。記錄：

```md
| 迴避場景 | 害怕什麼 | 實際風險 | 可嘗試的低強度接近步驟 | 是否需要專業支援 |
|---|---|---|---|---|
```
