# Weekly Review Module

> Layer ID: `MHE.REVIEW.WEEKLY`  
> Purpose: 每 7 天整理資料，不做診斷，只做 pattern review。

---

## 1. Weekly Review Questions

```txt
1. 今週最常出現的情緒是什麼？
2. 最強的一次情緒事件是什麼？
3. 最常見 trigger 是什麼？
4. 最常見身體反應是什麼？
5. 最常見腦中句子是什麼？
6. 最常見自保反應是什麼？
7. 哪一個 grounding / coping 方法最有效？
8. 有沒有 Red / Orange 風險？
9. 下週只想調整哪一件小事？
```
